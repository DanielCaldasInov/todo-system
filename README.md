# 📝 To-Do App (Task Management System)

A modern, minimalist, and highly efficient Task Management System built as a technical challenge. It allows users to manage daily activities, set priorities, track progress via checklists, and receive simulated daily reminders.

## ✨ Features

- **Authentication:** Secure login and registration.
- **Task Management (CRUD):** Create, read, update, and delete tasks.
- **Interactive Checklists:** Add dynamic subtasks to any task and toggle their completion status.
- **Advanced Filtering:** Filter tasks by Status (Pending/Completed), Priority (Low/Medium/High), and Due Date.
- **Responsive UI & Dark Mode:** A beautiful, accessible interface powered by Tailwind CSS v4, fully responsive for mobile devices, featuring a seamless Dark/Light mode toggle.
- **Automated Reminders (CRON):** A scheduled console command that simulates sending email notifications for tasks due tomorrow by writing to the application log.

## 🛠️ Tech Stack

- **Backend:** Laravel 12 (PHP)
- **Frontend:** Vue 3 (Composition API) + Inertia.js
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide Vue Next
- **Database:** SQLite (or MySQL)
